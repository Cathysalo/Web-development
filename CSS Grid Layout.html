CSS Grid Layout

This page uses CSS Grid to create a responsive layout. As you resize the screen, the layout will adjust from three columns to two or even one, depending on the screen size.

What is CSS Grid?

CSS grid creates complex responsive web design grid layouts more easily and consistently across browsers.

Grid Image Example

This image is inside a grid item and will adjust its size to fit the grid container.

Responsive Design

The grid layout is responsive, meaning it automatically adjusts based on screen size. It starts as a 3-column layout on large screens, changes to 2 columns on medium screens, and stacks into a single column on small screens like phones.

Another Image

Grid allows easy management of multiple types of content, such as images and text.

Settings

Settings & their effects include:

  • Grid-template-columns: Defines the number and size of columns in the grid. Example: grid-template-columns: repeat(3, 1fr); creates 3 equal columns.
  • Grid-gap: Controls the space between grid items. Example: gap: 20px; creates 20px of space between each item.
  • Media queries: Adjust the grid layout for different screen sizes.

Key Features

Key features of CSS Grid include:

  • Auto-placement: Grid items are automatically placed in the next available cell unless explicitly positioned.
  • Responsive design: Media queries adjust the grid layout dynamically based on screen width.
  • Aligning items: Use align-items or justify-items to center, stretch, or align content within grid cells.
  • Span items: Use grid-column or grid-row to make items span multiple columns or rows.
